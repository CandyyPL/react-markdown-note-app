# React Markdown Note App

This is a full stack note taking app build with Vite and React on frontend and Spring Boot with PostgreSQL on backend.

**Local Storage version** (without backend) is hosted on github pages [here](https://candyypl.github.io/react-markdown-note-app/).

## Try it yourself

Application is containerized with Docker for ease of use and general compatibility.

Images for frontend and backend are available to download from [my docker hub page](https://hub.docker.com/repositories/candyypl).

Application ports:

- frontend runs on port 3000
- backend runs on port 8080
- postgres runs on port 5432
